브라우저의 렌더링 과정에 대해서 상세하게 설명
예를 들어 "naver.com" 접속시에 해당 사이트 서버를 찾아간다. DNS 서버연결 -> 여기서 주소에 https 붙어있다면 https 방식으로 통신 -> 서버의 대부분의 기본파일인 index.html 파싱 -> DOM 트리 생성 -> CSS 파싱 -> DOM트리 완성 ->
  • 1. 주소창에 구글 입력
  • 2. 구글 서버로 찾아간다
    • 3. *DNS가 연결해줄 곳을 찾음 4. 서버에서 HTML 파일을 클라이언트로 보냄 5. HTML 파일 파싱 및 DOM Tree 생성 6. link 태그를 만나 css 파싱 및 CSSOM 트리 생성 DOM , CSSOM 합쳐 Render Tree 생성 (8. JavaScript를 만나면? HTML파서는 JS 코드를 실행하기 위해 파싱 중단 JS 엔진실행 및 JS코드 파싱)
